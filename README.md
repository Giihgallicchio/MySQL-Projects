# | MySQL - Estudos e Projetos

Repositório com exercícios e práticas desenvolvidas durante meus estudos de banco de dados utilizando MySQL.



## | Sobre

Este repositório reúne atividades focadas no aprendizado de conceitos fundamentais de banco de dados, como criação de tabelas, manipulação de dados e consultas SQL.

Os exercícios foram desenvolvidos com o objetivo de entender a estrutura e o funcionamento de bancos de dados relacionais.



## | Objetivo

- Aprender comandos SQL  
- Entender a estrutura de bancos de dados relacionais  
- Praticar criação e manipulação de tabelas  
- Desenvolver consultas para extração de dados  



## | Tecnologias utilizadas

- MySQL  



## | Conteúdos estudados

Durante os exercícios, foram abordados:

- DDL (Data Definition Language)  
- Criação de tabelas  
- Inserção de dados (`INSERT`)  
- Consultas (`SELECT`)  
- Atualização de dados (`UPDATE`)  
- Exclusão de dados (`DELETE`)  
- Limpeza de tabelas (`TRUNCATE`)  



## | Estrutura do Projeto

- `DDL.sql` → Criação de tabelas  
- `Criando a tabela cursos.sql` → Estrutura de tabela  
- `Cadastro de pessoas.sql` → Inserção de dados  
- `Adicionando Valores.sql` → Inserção de registros  
- `Select.sql` → Consultas básicas  
- `Select Parte 2.sql` → Consultas intermediárias  
- `Select Parte 3.sql` → Consultas mais avançadas  
- `Modificando o banco de dados Cadastro.sql` → Alterações no banco  
- `Update, Delete e Truncate.sql` → Manipulação de dados  



## | Destaque

Este repositório demonstra a prática dos principais comandos SQL utilizados no dia a dia de desenvolvimento com banco de dados.



## | Evolução

Este projeto representa minha evolução no aprendizado de MySQL, desde comandos básicos até consultas mais estruturadas.
